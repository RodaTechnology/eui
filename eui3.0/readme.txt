=================== 本地部署说明 =====================
1. 解压文件。
|
|---- boot.js 加载静态资源文件
|---- assets  EasyUI相关文件
|---- modules 模块
|---- tools   第三方工具
|
2. 将解压缩得到的文件放在本地resources文件夹下。
3. 打开header.ftl文件，引入resources下面的boot.js即可


=================== 远程部署说明 =====================
1. 解压文件。
|
|---- boot.js
|---- assets
|---- modules
|---- tools
|
2. 将解压缩得到的文件上传到远程服务器。
3. 打开header.ftl文件，引入远程服务器的boot.js即可
