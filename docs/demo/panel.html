扩展的属性

该扩展增加定义了如下属性：

Name Type Description Default
plain boolean 表示 easyui-panel 面板简单模式 false
fieldset boolean 表示 easyui-panel 面板类似fieldset模式 false
fitWidth boolean 表示 easyui-panel 自适应宽度，一般用于不能自适应高度，但是要自适应宽度的布局情况下 false
minWidth number 表示 easyui-panel 面板的最小宽度 0
maxWidth number 表示 easyui-panel 面板的最大宽度 10000
minHeight number 表示 easyui-panel 面板的最小高度 0
maxHeight number 表示 easyui-panel 面板的最大高度 10000
iniframe boolean 表示 href 加载的远程页面是否装载在一个 iframe 中 false

扩展的事件

该扩展增加定义了如下事件：

Name Parameters Description
onResize width,height 重写 onResize 事件，用于扩展四个新增属性 minWidth、maxWidth、minHeight、maxHeight 的功能。

扩展的方法

该扩展增加定义了如下方法：

Name Parameters Description Returns
inLayout none 判断当前 easyui-panel 是否为 easyui-layout 的 panel 部件。
 如果当前 easyui-panel 是 easyui-layout 的 panel 部件，则返回 true，否则返回 false。
inTabs none 判断当前 easyui-panel 是否为 easyui-tabs 的选项卡。
 如果当前 easyui-panel 是 easyui-tabs 的 panel 部件，则返回 true，否则返回 false。
isWindow none 判断当前 easyui-panel 是否为 easyui-window 组件。
 如果当前 easyui-panel 是 easyui-window 的 panel 组件，则返回 true，否则返回 false。
isDialog none 判断当前 easyui-panel 是否为 easyui-dialog 组件。
 如果当前 easyui-panel 是 easyui-dialog 的 panel 组件，则返回 true，否则返回 false。
iframe none 获取当前在 iniframe: true 时当前 panel 控件中的 iframe 容器对象。
 如果 inirame: false，则该方法返回一个空的 jQuery 对象。
refresh href 重写 easyui-panel 控件的 refresh 方法，用于支持 iniframe 属性。
 返回当前 easyui-panel 的 jQuery 对象。
header none 重写 easyui-panel 控件的 header 方法，支持位于 easyui-tabs 中的 tab-panel 部件获取 header 对象。
 如果该 panel 位于 easyui-tabs 中，则该方法返回 easyui-tabs 的 div.tabs-header div.tabs-wrap ul.tabs 中对应 tab-panel 的 li 对象。
setTitle title 重写 easyui-panel 控件的 setTitle 方法，支持位于 easyui-tabs 中的 tab-panel 部件设置 title 操作。
 返回当前 easyui-panel 的 jQuery 对象。
